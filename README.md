# Movie Finder Project

A web application that allows users to search and filter movies based on their ratings. The project is designed with a clean, user-friendly interface for easy navigation and interaction.

## Features

- Search bar for quick movie searches.
- Filter options to sort movies by ratings.
- Responsive design for optimal viewing on all devices.
- Clean and modern user interface with hover and focus animations.

## Technologies Used

- **HTML** - For the structure of the web page.
- **CSS** - For styling and layout.
- **JavaScript** - For interactivity and dynamic filtering.
